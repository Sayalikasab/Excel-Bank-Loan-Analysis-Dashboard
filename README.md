# Excel-Bank-Loan-Analysis-Dashboard

## Project Overview
This Excel dashboard analyzes bank loan data using interactive charts, Pivot Tables, KPI Cards, and Slicers.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning

## Dashboard KPIs
- Total Loan Amount
- Total Payment
- Verified Loans
- Average Interest Rate

## Reports
1. Year Wise Loan Amount Stats
2. Grade & Sub Grade Wise Revol_Bal
3. Verified vs Non Verified Total Payment
4. State Wise & Month Wise Loan Status
5. Home Ownership vs Last Payment

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Design
- Data Visualization
- Excel Reporting
